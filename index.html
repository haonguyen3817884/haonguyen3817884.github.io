Project Ideas

Virtual Online Investment Platform

Overview: This project is about making a virtual online investment platform.The website allows people to invest their money virtually. The purpose of this project is to let people have a chance to practice analyzing the financial information and investing. This project contains two parts ( IT part and financial part ). The IT part involves creating and maintaining the website. This includes coding, debugging and optimizing. The financial part involves collecting financial data and forming financial formulas.

Motivation: There are several reasons why this project will be very useful. Firstly, it helps young people develop a good habit. According to many articles, a lot of young students today do not have a habit of saving money. They always spend money on unnecessary things like new shoes. This is mainly because they do not find any reasons to save their money or they do not know what benefits saved money will bring to them. Therefore, having such websites is needed. By earning profit from investments they will have more reasons to save their money. Secondly, it is a safe place for financial students to practice what they learn. Investing in real financial markets may be too risky for many students. Therefore, using virtual money to invest will keep them safe from making wrong decisions.

Description(1): This website will function as real online investment platforms. Firstly, there will be a variety of financial markets on the website. Not only the stock market, it also provides other common financial markets like foreign exchange or gold market. therefore, Investors will have more choices to invest. They will be able to choose one or many markets at the same time. Secondly, the user account will work as real banking accounts. It will store all the personal information as well as the financial information privately. All the profit or loss from investment will also be recorded and added into customer’s account. Thirdly, virtual money but real experience. All the financial information on the website like the stock price or the company’s name are taken exactly the same from real financial markets. This will make customer’s experience on the web more realistic.

Description(2): Not only that, It is also a good place to learn and practice financial analysis. Firstly, there will be a place for users to make financial analysis. A platform will be added to help them create their own analysis. This platform will have a form of notebook and the record of the analysis will last until the users delete it. Secondly, it is easy to do simple financial analysis on the website. For those who are new to the financial world, finding information as well as formulas to make analysis can be very difficult. There are a lot of information and formulas to collect. To make it more convenient, all the information and basic financial formulas are fully provided on the website. Besides, learning basic financial formulas will be easier. For each formula, there is always a paragraph below to explain the meaning of it. The website also includes funny exercises to help people who have not studied finance before to get used to reading and analyzing financial information. Finally, financial students can perform complex analysis or expand their current analysis. The website will have a platform to store financial data and it will update new financial information. Students can use them to keep updating their analysis.

Tools and Techonologies: To make this project work, personal computers, financial software and mainframe computers are required. Personal Computers are used to create and manage the website by using programming languages like HTML, PHP or Java. Financial software refers to two types of software ( financial management and financial calculation ). Financial management software allows the website to do accounting functions like tracking the budget, recording transactions and organizing investment portfolios. Financial calculation software helps the website perform calculations based on financial formulas to let the user measure the profit or loss of investment. A mainframe is a big computer which manages the database. It is mostly used to maintain the website by storing and transferring both financial and user data.

Skills Required: There are a few skills required to implement this project successfully. One of the most important skills that need to be obtained at first is problem-solving skills. In order to turn this project idea into reality, there will be a lot of problems that need to be dealt with. Therefore, the better you are good at this skill, the better project implementation will be. The next important requirement is mastering front-end programming languages like HTML or Java. these languages will help you create the website and make applications on this. Having knowledge of finance is also required when you do this project. In order to build the financial software, you must have a deep understanding of financial data and formulas to write codes. The final requirement is an in-depth knowledge of back-end programming languages like MySQL or Python. It is so important because mastering those languages will help you manage the database system.

Outcome: If this website is really created, it will bring benefits for both education and financial industry. For education, it provides a new way for financial students to learn practical knowledge. Instead of practicing by doing exercises in class, they can create an account on the website and use virtual money to learn investing. For financial industry, a new method of checking analysis will be created. By making analysis to do investing on this website, investors can check whether their analysis is right or not.
